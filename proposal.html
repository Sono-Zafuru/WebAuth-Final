Portfolio Website Proposal

Website's Purpose

I'm creating a portfolio website to show off my skills as both a programmer and an artist. By coding the website using what I've learned through the Web Authoring course and from my own research, I'll be able to use this as a showcase of said skills for potential clients and employers.

Along with the home page, there will be dedicated pages for the following:

  • About Me, which includes personal anecdotes and additional education information
  • Portfolio, for displaying my work and other projects
  • Resume, summarizing the portfolio website in one page while listing other work experience
  • Contact, which has my contact info including phone number and email

Target Audience

Truthfully, the audience I intend to reach are potential employers and/or clients in game design and animation, so I know that I'll need to bulk up this portfolio with projects related to those.
